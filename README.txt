# auction

Streamlabs Chatbot script to run auctions using your Streamlabs currency.

Contribute to the script at https://github.com/aseroff/auction/

v 1.1.0 :

More string customization, ensure bidders have enough currency, include currency name.

v 1.0.0 Initial release:

Users with mod and higher permissions can start an auction with `!auction [thing to be auctioned]`.
Chatters can type `!bid [value]` to bid on the auction. When there's a winner, the winning bid is withdrawn from their currency.
